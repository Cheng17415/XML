Propiedades CSS: Bordes y Fondo

text-shadow: Sombra aplicada al texto. Aquí se usa text-shadow: 4px 4px 6px #aaa;.
background-image: Imagen como fondo. Aquí se usa background-image: url(...);.
background-repeat: Controla la repetición del fondo. Aquí se usa no-repeat.
background-position: Posiciona el fondo en el centro. Aquí se usa center.
background-attachment: Fija el fondo al viewport. Aquí se usa fixed.
border: Borde sólido azul de 5px. Aquí se usa border: 5px solid blue;.
border-style: Estilos variados (punteado, sólido, discontinuo, doble, surco). Aquí se usa border-style: dotted solid dashed double groove;.
border-width: Grosor del borde. Aquí se usa border-width: 10px;.
border-color: Define colores diferentes para cada lado. Aquí se usa border-color: red green blue yellow;.
border-top/bottom/left/right: Diferentes estilos de borde por lado. Aquí se usan estilos mixtos.
border-radius: Hace los bordes redondeados o circulares. Aquí se usa border-radius: 50%;.
